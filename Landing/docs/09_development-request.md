# 개발 요청사항

## 작업 목표
정찰제 홈페이지 제작 서비스를 소개하는 원페이지 랜딩페이지를 제작한다.

## 개발 기준
- Next.js 기반
- 모바일 반응형 필수
- 원페이지 구성
- 섹션별 컴포넌트 분리
- 가격표 카드 UI 구현
- 상담 CTA 버튼 반복 배치
- SEO 메타 태그 적용
- 추후 수정이 쉽도록 콘텐츠 데이터 분리 권장

## 우선순위
1. 모바일 화면 완성도
2. 가격표 가독성
3. 상담 버튼 접근성
4. 빠른 로딩
5. 유지보수 쉬운 구조

## 주의사항
- 추상적인 디자인회사 홈페이지처럼 만들지 말 것
- 가격과 상품이 명확하게 보이도록 할 것
- 과한 애니메이션은 사용하지 말 것
- 첫 화면에서 무엇을 파는지 즉시 보여줄 것
