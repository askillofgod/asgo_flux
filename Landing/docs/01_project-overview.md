# 프로젝트 개요

## 사이트 목적
당근 광고를 통해 유입된 소상공인, 자영업자, 1인사업자에게 정찰제 홈페이지 제작 상품을 소개하고 상담 문의를 받는 원페이지 사이트를 제작한다.

## 핵심 방향
- 감성적인 디자인회사 소개가 아니라 상품 판매형 페이지
- 가격이 먼저 보이는 정찰제 구조
- 빠른 제작, 쉬운 상담, 명확한 패키지 강조
- 모바일 중심 구성

## 타깃
- 소상공인
- 자영업자
- 1인사업자
- 병원, 학원, 공방, 식당, 미용실, 스타트업
- 광고용 랜딩페이지가 필요한 업체

## 사이트명
ASOG 정찰제 홈페이지 제작소
